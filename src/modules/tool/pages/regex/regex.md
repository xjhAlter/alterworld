###入门

``` bash
    /ab/        // 精确匹配
    /ab+/       // 模糊匹配
    /[abc]/     // 匹配a、b、c其中一个
    /[a-c]/     // 匹配a、b、c其中一个
    /[^abc]/    // 不匹配a、b、c中任意一个
```

###量词

```bash
    {m,}        // 最少出现m次
    {m,n}       // 最少出现m次,最多出现n次
    ?           // 匹配0次或1次
    +           // 最少出现1次, 等价于{1,}
    *           // 匹配任意次数, 等价于{0,}
```

###位置匹配

```bash
    ^           // 匹配开头
    $           // 匹配结尾
    
```

###反向引用

```bash
    /(a)b\1/    // 可匹配aba,在配对'',""等字符时非常有效
    \10         // 表示第十个捕获组,需要表示第一个捕获组+0时要用/(\1)0/
```

```bash
    ?:          // 非获取匹配,匹配冒号后的内容但不获取匹配结果,不进行存储
```
